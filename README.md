# ever-photobook

## 시작하기

### 환경

- Bun 1.2.6 이상
- Node.js 21.5.0

### 의존성 설치 및 개발 서버 실행

```
./init.sh

bun run dev
```

### 환경 구성 테스트

```
bun run format-lint
```

> 커밋 이전에 위의 명령어를 실행하고 커밋을 진행

### 이미지 배치
`/public/images` 위치에 '연-월-일-번호' 형식으로 이미지명을 지정

```
(예시)
...
├── public       # 정적 자산 파일
│   ...
│   └── images
│       ├── 2025-11-12-001.jpg
│       ├── 2025-11-12-002.jpg
│       ├── 2025-11-13-001.jpg
│       ├── 2025-11-13-002.jpg
│       ├── 2025-11-14-001.jpg
│       ...
│       └── 2025-11-17-999.jpg
└── src 
```
